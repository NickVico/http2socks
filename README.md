# socks2http #

a daemon transport http request to socks proxy


    Usage of ./http2socks:
      -addr string
            address listen to (default "127.0.0.1:8081")
      -help
            show this help
      -socks_addr string
            socks server addr (default "192.168.56.1:8080")
      -socks_proto string
            socks protocol type, [socks5|socks4|socks4a] (default "socks5")
